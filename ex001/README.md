# Verificador de Idade

Projeto simples feito para treinar JavaScript, HTML e CSS.

O sistema verifica a idade digitada pelo usuário e informa se ele é maior ou menor de idade.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Manipulação de DOM
- Estruturas condicionais (if / else)

## Como funciona

1. O usuário digita sua idade
2. Clica no botão de verificação
3. O sistema analisa a idade usando JavaScript
4. O resultado aparece na tela com uma cor diferente dependendo da idade

## Aprendizados com este projeto

- Capturar valores de inputs
- Converter valores usando `Number()`
- Manipular elementos da página com JavaScript
- Alterar estilos CSS dinamicamente
- Utilizar eventos (`addEventListener`)
- Utilizar condicionais `if / else`
